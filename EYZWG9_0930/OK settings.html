A Visual Studio Code alapbeállításai, és bővítményei.

Alapbeállítások

Téma kiválasztása a File > Preferences > Color Theme menüpontban, ahol lehet válogatni az előre elkészített témák között.
Bővítmények telepítése bal oldali sávon a Extensions ikonra kattintva érhetőek el.
Automatikus mentés engedélyezése a files.autoSave beállítással.
